Features

So what can Plex Requests .Net do?

Usenet applications Integration

Plex Requests.Net is compatible with Sonarr, Sickrage, Couchpotato and Headphones!

Cross Platform

Running Windows, Linux, OSX or some other OS? It's cool, we got it covered.

Plex Integration

We actually check Plex to see if there is content present before actually allowing the user to request!

Responsive

You can easily request new content for Plex and administer your requests from mobile devices anywhere in the world!

Need any more reasons?!

Seriously... If you need some more reasons to use Plex Requests .Net then just keep on reading!

Notifications

We currently support Email, Pushbullet and Pushover notifications. Get notified for new requests, issues a user has reported. Even users can now get notifications once a request is available!

It's a tiny application

We have been able to get a nice small memory footprint when using Plex Requests .Net, Even with all of it's background processing it's still nice and small!

Excellent design architecture

We have the ability to add new features quickly and efficiently, the turnaround for new features/bug fixes is fantastic.
